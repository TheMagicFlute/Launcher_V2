# 文件结构树

```text

Launcher_V2
├── Launcher_x**.exe   启动器
└── Profile            由启动器生成的配置文件目录
   ├── Config.json         设置选项
   ├── TrainingMission.xml 赛道对决记录列表
   ├── TimeAttack.log      计时练习记录列表
   ├── Favorite.xml        星标道具列表
   ├── FavoriteTrack.xml   星标赛道列表
   ├── RandomTrack.xml     随机赛道列表
   ├── Item.xml            道具列表
   ├── NewKart.xml         卡丁车列表
   ├── KartSpec.xml        卡丁车数据列表
   ├── FlyingPetSpec.xml   飞行宠物数据列表
   ├── PartsData.xml       X, V1部件列表
   ├── PlantData.xml       9以及之前部件列表
   ├── LevelData.xml       车辆升级加点列表
   └── TuneData.xml        车辆改装列表

```
