    公告¶

    • 某些平台上有人将本启动器打包并进行售卖，价格由 5-30 RMB 不等。

      • 该类商户未提供维护者、程序集来源等信息，经营所得资金也并未用于支持本项目的开发。
      • 他们一般会建议用户关闭杀毒软件以避免误报，但这些程序集的完整性无法得到保证，可能会被植入恶意代码。
      • 更有甚者将有“Yany”角色名以及“本单机启动器完全免费”走马灯横幅字样的图片作为宣传图，但仍有人视而不见，上当受骗。

      重申：本单机版完全免费！

      请不要支持任何形式的二次打包和售卖。他们不会为启动器的使用提供任何帮助和支持，且有可能会给你的电脑带来安全隐患。

      希望各位车友提高警惕，擦亮双眼，捂好钱袋子，不要下载任何来路不明的文件！

    • 自 250809 开始，Yany 仓库下舍弃了对 32 位操作系统架构( x86 )的支持。

      目前，Yany 仓库的 Release 中，只发布 64 位操作系统架构( x64 )， 以及 Arm64 操作系统架构( arm64 )的启动器。

      如果你的操作系统架构为 32 位( x86 )：

      • 在 安装 中根据提示自行下载源码并编译
      • 接收 TheMagicFlute Release 中提供的 32 位启动器，并一直使用该频道接受更新。

    简介¶

    你好，欢迎来到本跑跑卡丁车单机启动器的帮助文档！ 该文档位于 TheMagicFlute Fork 下，你可以点击右上方的两个按钮进行查看或者 Fork 后编辑。

    本启动器完全免费，且不包含任何广告，只有 GitHub Release 作为唯一下载途径，请不要下载任何来路不明的文件！ 本单机启动器由安全版本剥离并进行开发，并使用 GitHub Action 的 CI/CD 进行发布。 没有任何病毒，请放心使用。

    这是一个基于 .NET 8.0 的中国跑跑卡丁车单机启动器，修改自 LAON 制作的国服启动器，并 fork 自 Yany 大佬，修改并添加了新功能。

    TheMagicFlute 仓库 会及时同步 Yany 仓库 中做出的修改，并发布更新。
